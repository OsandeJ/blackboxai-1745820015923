# Sistema de Gestão

## Project Overview

Sistema de Gestão is a web-based management system designed to streamline the tracking and management of products, customers, and orders. The application features a user-friendly interface with an intuitive sidebar navigation that allows users to quickly switch between different sections including products, customers, and orders. With functionalities for adding, editing, and deleting records, this system aims to simplify everyday business operations.

## Installation

To set up this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://your-repository-url.git
   ```

2. **Navigate into the project directory**:
   ```bash
   cd sistema-de-gestao
   ```

3. **Open `index.html` in your web browser** to view the application:
   ```bash
   open index.html
   ```
   Alternatively, you can use a local server for better performance (e.g., using Live Server in Visual Studio Code).

## Usage

Upon opening the application, you'll find a sidebar with navigation links for "Produtos", "Clientes", and "Pedidos". Here’s how to use the application:

- Click on the **Produtos** link to view a list of products where you can perform actions to add, edit, or delete products.
- Click on the **Clientes** link to manage your customer records, enabling you to add, edit, or delete customer information.
- Click on the **Pedidos** link to view and manage orders, with options to add new orders, view details, and print invoices.

## Features

- **Responsive Design**: The application is responsive and works well on various devices.
- **Dynamic Navigation**: Users can navigate between sections easily via an intuitive sidebar.
- **Data Management**: The system allows addition, editing, and deletion of products, customers, and orders.
- **Action Buttons**: Each section includes action buttons for quick access to functionalities like adding new records.
- **Interactive Tables**: Tables provide an overview of data with a hover effect for easy readability.

## Dependencies

This project does not have any external dependencies specified in a `package.json` file as it solely relies on HTML, CSS, and JavaScript.
